# Welcome to the Mini Message Board

This is an Express App build with the following tech stack:

- Express.js - Backend
- EJS - Templating language
- CSS - Styling
- Express Validator - Sanitzation of Form elements

This is a project for The Odin Projects Node course. The intention of this project is to gain familiary with Express, and using the MVC pattern, Express validation for sanitization of form data. I have an on-going todo list of items I would like to add to this project to make it a worthy portfolio project.
